## Initial Product Direction

HomeScout Copilot should be a property and area comparison assistant, not a regulated mortgage advice engine.

Initial data-source ideas:

- Police.uk API for street-level crime data.
- HM Land Registry open data for price paid and house price context.
- GOV.UK school performance data for school comparison context.
- OpenStreetMap Overpass API for amenities.
- TfL Unified API for London transport and commute analysis.

Do not modify files in this folder after creation. If a source changes, create a new raw file with the date in the filename and link it from `wiki/index.md`.
